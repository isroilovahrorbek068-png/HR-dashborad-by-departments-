## ğŸ“Š Project Overview

This interactive HR Analytics Dashboard was built using **Power BI** to provide insights into workforce composition, employee status, and organizational trends.

The dashboard allows stakeholders to analyze:
- Employee distribution
- Gender diversity
- Promotion eligibility
- Years of service
- Job levels
- Distance from the office  

The goal of this project is to transform **raw HR data into meaningful, actionable insights** that support data-driven decision-making and effective workforce planning.
